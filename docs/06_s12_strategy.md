## 2. HAKO-NIWA の差別化ポジショニング（マーケ核）

- **「その場の新たな出会い」**(殿FB#9): 1対1アプリ疲れ・サブスク疲れ・メッセージ往復疲れへの解。**会うことに全集中**(アプリ内チャット無し)。
- **安心の三本柱**: ①本人確認必須(年齢確認) ②グループ(2-4人ずつ計6人)で気まずさ・危険を希釈 ③ドタキャン罰金で真剣な人だけ。
- **おまかせ体験**: 会場は運営手配・前日夜に確定通知。ユーザーは「行くだけ」。
- **従量・成立課金**: サブスク不要。男性は参加が決まった時だけ¥2,000(初回無料)・女性無料。不成立は非課金。**「参加費」と明記・飲食代は別**(殿FB#3)。
- **写真より"場"**(殿FB#8): 顔写真詐欺の土俵に乗らない。プロフはアイコン＋職業＋年齢の軽量。会って人柄で。

→ LP・コピー・価値訴求はこの軸で全編統一(「写真詐欺がない」は撤去=写真使わないため。代わりに「会うことに集中・その場の出会い」)。

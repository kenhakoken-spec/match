# LINE設定をClaudeに操作させる — 殿が戻ったら、これだけ

殿が「自分の今のChromeでやりたい」を選択 → **Chromeをデバッグモードで起動し直すと、私がそのChromeを操作できます。**
（通常のChromeは外部接続を受け付けない仕様なので、一度だけ起動し直しが必要です。）

## 手順（戻ったらこれを実行）

1. **既存のChromeを全部閉じる**（ウィンドウを全部）。
2. エクスプローラーで `C:\tools\matching-app\tools\start-chrome-debug.bat` を**ダブルクリック**。
   - 黒い画面が出て、Chromeがデバッグモードで開きます（LINEコンソールが表示される）。黒い画面は閉じてOK。
3. その Chrome で **LINE にログイン**（ここは殿だけができる作業）。
4. 私（Claude）に「**ログインした**」と伝える。

そのあと私が:
- そのChromeに接続して、プロバイダー作成 → LINE Loginチャネル → LIFF追加 のフォーム入力を**代行**
- 各ステップでスクショを撮って殿に見せる
- **最後の「Create／発行」ボタンだけ殿が押す**（最終承認）

## 技術メモ（私用）
- WSL は **NATモード**、Windowsホスト = `172.27.32.1`（既定GW）。
- bat は `--remote-debugging-address=0.0.0.0` で待受 → WSLから `http://172.27.32.1:9222/json/version` で接続を試す。
- NAT越えで繋がらない場合 → **画面SSガイド方式**（`docs/setup/external-setup.md` の実画面SS対応表）に切替（確実）。
- 接続できたら playwright `connectOverCDP("http://172.27.32.1:9222")`。

## 急ぎでない
LINE実キーが要るのは S3（通知）以降。今はモックで S1〜S6 を作り進めるので、**ログインは殿の都合の良い時で大丈夫**です。

## U-12 本人認証（身分証アップロード）★ 新規 / 必須ゲート（STEP0）

公的身分証を提出。**未認証は枠応募不可**。PII につき安心感のある説明を添える（master_plan §8/§9）。

```
┌─────────────────────────────┐
│ ←  本人確認                  │
├─────────────────────────────┤
│  安心してご利用いただくため   │
│  本人確認をお願いします。     │
│  ・年齢確認（18歳以上）を兼ねます│
│  ・画像は確認後に削除します   │  ← PII最小化の明示（信頼感）
│  ・公開されることはありません  │
│                             │
│  身分証の種類                │
│  ( 運転免許証 )( マイナンバーカード )│  ← docType（顔写真付き公的書類）
│  ( パスポート )              │
│                             │
│  画像をアップロード          │
│  ┌─────────────────────┐    │
│  │     ▢ 表面を撮影/選択 │ ＋ │  ← カメラ起動 or 端末から選択
│  └─────────────────────┘    │
│  ┌─────────────────────┐    │
│  │     ▢ 裏面（必要時）  │ ＋ │
│  └─────────────────────┘    │
│  ・氏名 / 生年月日 / 顔が     │
│   はっきり写るように          │  ← 撮影ガイド
│                             │
├─────────────────────────────┤
│   [  提出する  ]             │  ← 必須画像が揃うと活性
│   ( あとで ) ※応募はできません │  ← スキップ時は応募ブロックを明示
└─────────────────────────────┘
※ IdentityVerification=pending を作成。画像はアクセス制限ストレージへ。
※ 提出後 → U-13 審査中へ。
```

├─────────────────────────┤    ├─────────────────────────┤    ├─────────────────────────┤
│        ◷                │    │        ◆認証済          │    │        ⚠                │
│  確認中です              │    │  確認が完了しました       │    │  確認できませんでした     │
│                         │    │                         │    │                         │
│  通常 1〜2 営業日で      │    │  ご協力ありがとう        │    │  理由:                   │
│  完了します。            │    │  ございます。            │    │  ・画像が不鮮明でした     │  ← 却下理由（運営入力）
│  完了したらLINEで        │    │                         │    │  ・顔写真が確認できません  │
│  お知らせします。        │    │  プロフィールを          │    │                         │
│                         │    │  登録しましょう。        │    │  お手数ですが            │
│  ( ホームへ )           │    │  [ プロフィール登録へ ]  │    │  もう一度お願いします。   │
